# 🐧 Linux Study Lab: De Fundamentos ao Profissional

Este repositório contém uma trilha de aprendizado prática sobre administração de sistemas Linux, organizada em missões, exercícios por níveis e cenários de resolução de problemas reais.

## 📂 Estrutura do Repositório

O conteúdo está organizado da seguinte forma:

- `missões/`: Desafios estruturados simulando tarefas reais de um analista.
- `exercícios/`: Práticas divididas por níveis (Fundamentos, Processos, Hardware).
- `cenários/`: Estudos de caso para resolução de problemas (Troubleshooting).
- `scripts/`: Exemplos de automação e comandos úteis.

## 🚀 Como usar este guia

Cada diretório contém um arquivo `README.md` com as instruções daquela seção. Recomenda-se seguir a ordem:
1. **Missões** (Para uma visão geral prática)
2. **Exercícios Nível 1 a 3** (Para consolidar comandos)
3. **Cenários Nível 4** (Para testar habilidades de diagnóstico)

---
*Este material foi organizado para facilitar o estudo e a consulta rápida de comandos essenciais de gestão de identidade, processos e hardware.*
